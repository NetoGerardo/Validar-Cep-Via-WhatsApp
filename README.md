# Validador de Cep Via WhatsApp

## Instalação
A primeira coisa a se fazer é instalar os módulos:
```bash
npm install
```
Agora está tudo pronto para executar!

## Como executar a Api
Abra o prompt de comando e execute:
```bash
node app.js
```

## Como executar o Validador Via WhatsApp
Abra o prompt de comando e execute:
```bash
node teste.js
```
