# SauceDemo

## Versão do Python
3.10.2

## Instalação
A primeira coisa a se fazer é instalar o Selenium:
```bash
npm install
```
Agora está tudo pronto para executar!

## Como executar a Api
Abra o prompt de comando e execute:
```bash
node app.js
```

## Como executar o Validador Via WhatsApp
Abra o prompt de comando e execute:
```bash
node teste.js
```
